# 🚀 Root Portfolio

Portfólio digital interativo desenvolvido com React, Vite e Tailwind CSS.

## 🛠️ Tecnologias

- React 18
- Vite
- Tailwind CSS
- Framer Motion

## 🚀 Como executar

```bash
npm install
npm start
```
